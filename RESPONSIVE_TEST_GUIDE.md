## 🧪 测试方法

### 1. Chrome DevTools 测试
```
1. 打开 Chrome DevTools (F12)
2. 点击设备工具栏图标 (Ctrl+Shift+M)
3. 测试以下预设设备:
   - iPhone SE (375x667)
   - iPhone 12 Pro (390x844)
   - iPad (768x1024)
   - iPad Pro (1024x1366)
   - Desktop (1920x1080)
```

### 2. 响应式断点测试
手动调整浏览器宽度,测试关键断点:
- 320px (最小手机)
- 640px (sm 断点)
- 768px (md 断点)
- 1024px (lg 断点)
- 1280px (xl 断点)

### 3. 主题测试
- [ ] 在亮色主题下测试所有功能
- [ ] 切换到暗色主题
- [ ] 在暗色主题下测试所有功能
- [ ] 主题切换平滑无闪烁

### 4. 触摸设备测试
在真实移动设备上测试:
- [ ] 点击响应灵敏
- [ ] 滚动流畅
- [ ] 手势操作正常
- [ ] 没有误触问题

## 🐛 常见问题排查

### 样式未生效
1. 清除浏览器缓存
2. 重启开发服务器
3. 检查 Tailwind 配置

### 动画卡顿
1. 检查 GPU 加速
2. 减少同时触发的动画数量
3. 优化动画性能

### 移动端布局错乱
1. 检查视口 meta 标签
2. 验证响应式类名
3. 测试不同设备方向
